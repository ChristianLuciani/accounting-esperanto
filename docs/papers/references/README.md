## Key editorial note

The IFRS Foundation publishes jurisdiction profiles for **169 jurisdictions** (not 195).
Kontablo maps **195 UN-recognized sovereign jurisdictions** — these are the total sovereign
entities Kontablo covers, not all of which have formal IFRS Foundation profiles.
The distinction is clarified in the preprint introduction and appendix.
